# 🎨 AI Resume Generator - 105+ Templates

A professional resume generator with 105+ unique templates, built with Streamlit.

## Features

- ✅ 3 Unique Styles with 35 variations each (105+ templates)
- ✅ Photo Upload functionality
- ✅ PDF Download (using wkhtmltopdf)
- ✅ Word Document Download with theme colors
- ✅ All Sections: Education, Experience, Certifications, Achievements, Languages, Personal Details
- ✅ Add/Delete entries in real-time
- ✅ Live Preview

## Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/resume-generator.git

# Navigate to project directory
cd resume-generator

# Install dependencies
pip install -r requirements.txt

# Install wkhtmltopdf (for PDF generation)
# Ubuntu/Debian:
sudo apt-get install wkhtmltopdf
# Mac:
brew install wkhtmltopdf
# Windows: Download from https://wkhtmltopdf.org/downloads.html

# Run the app
streamlit run app.py
